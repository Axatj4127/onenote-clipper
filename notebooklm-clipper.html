NotebookLM to OneNote

v7 build - notebook sections + page picker

Sign in

Uses Microsoft Graph directly from this browser. The file must be opened through http://localhost:8080/notebooklm-clipper.html, not file:///.
v7 loads sections for the selected notebook. If OneNote/Graph returns an untitled section, use the local label buttons below to display the correct name in this tool.
Content will be inserted at the top of the selected page. The page list is populated after you choose a section and switch to Prepend mode.
